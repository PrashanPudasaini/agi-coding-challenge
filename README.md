## About the project

Simple CLI app that reads a csv file and processes them using pandas dataframe

## Prerequisites

* [Python 3.10.2](https://www.python.org/downloads/)
* Pandas
`pip install pandas`

## Installation

* Clone the repo
  `git clone https://github.com/PrashanPudasaini/agi-coding-challenge.git`

### Run the cli app

`python main.py`

### Run the test

`python test.py`
